## ğŸš€ Getting Started

### Prerequisites

- Flutter SDK (latest version)
- Firebase account and project setup
- Android Studio/VS Code with Flutter plugins
- Cloudinary account for image processing

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/CivikDesk.git
   cd CivikDesk
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Configure Firebase**
   - Add your `google-services.json` for Android in `android/app/`
   - Add your `GoogleService-Info.plist` for iOS in `ios/Runner/`
   - Update Firebase configuration in `lib/firebase_options.dart`

4. **Configure Cloudinary**
   - Update Cloudinary API keys in appropriate configuration files

5. **Run the app**
   ```bash
   flutter run
   ```

## ğŸ¯ Key Workflows

### User Journey
1. Sign up/Sign in with email or Google
2. Complete profile setup
3. Submit a complaint with details and images
4. Track complaint status in real-time
5. Receive notifications on status updates
6. View resolution and provide feedback

### Admin Journey
1. Secure login with admin credentials and special key
2. Review dashboard with pending complaints
3. Process complaints by department
4. Update status and add resolution details
5. Post resolved complaints to public feed
6. Generate reports and analytics
